# Spyfall (间谍) 🕵️‍♂️

Spyfall 是一款极具社交趣味的“身份隐藏”与“逻辑推理”类派对游戏。

---

## 📖 游戏规则

### 1. 核心目标
*   **间谍 (The Spy)**：隐藏身份，并在游戏结束前猜出当前所在的**地点**。
*   **特工 (The Agents)**：保护任务地点信息，同时识破间谍身份并指认间谍。

### 2. 游戏流程

#### 🛠️ **第一步：分发身份**
- 游戏分为多轮，每轮随机抽取一个**地点**。
- 除一名**间谍**外，所有玩家获得相同的地点信息，但每个人分配到的**相关职业**不同。
- 特工玩家需要根据职业设定进行角色扮演。

#### ❓ **第二步：问答环节**
- 玩家轮流向其他玩家提问。提问需隐晦：既要向队友暗示自己知道地点，又不能让间谍猜到。
- **轮转逻辑**：
    - 首轮随机玩家开始，后续轮次由上一轮间谍开始。
    - 提问者可以指定任意一名其他玩家回答一道问题（除了刚刚提问自己的玩家）。
    - A 提问 B -> B 回答 -> B 提问 C（B 不能立即反问 A）。
- **回合制**：每轮持续 T 分钟。

#### 🚨 **第三步：指控与揭发**
- **实时指控**：游戏过程中，任意玩家可发起指控（每轮仅一次机会）。
- **非匿名投票**：指控发起后进入投票环节，**所有投票均为实名/非匿名公开**。
- **全票一致**：若除被指控者外所有人投票通过，则身份揭晓，当轮结束。

#### 🔓 **第四步：间谍反击**
- 间谍可随时主动亮牌并猜测地点。
- **若猜对**：间谍立即获胜。
- **若猜错**：特工玩家获胜。

### 3. 结算逻辑

#### 🏆 **胜负判定**
*   **指认成功**：若投出的玩家确为间谍，特工方胜。
*   **指认错误**：若投出的玩家是特工，间谍胜。
*   **时间到**：进入“最终判定”阶段，每位玩家按顺序发起一次全票指控。若全员失败且未达成一致，间谍胜。

#### 💰 **积分系统**
| 达成条件 | 获胜方 | 分数分配 |
| :--- | :--- | :--- |
| **没有玩家被指控为间谍且全票通过** | 间谍 | 间谍 +2分 |
| **间谍猜对地点** | 间谍 | 间谍 +4分 |
| **玩家指控错误** | 间谍 | 间谍 +4分 |
| **玩家指控成功** | 特工 | 所有特工 +1分，指控发起者 +2分 |

## ⚙️ 游戏设定 (Configuration)

| 配置项 | 推荐值 | 说明 |
| :--- | :--- | :--- |
| **玩家人数** | 3 - 8 人 | 建议 5 人以上体验最佳 |
| **最大轮数** | 3 - 10 轮 | 总游戏时长控制 |
| **单轮时长** | 5 - 15 分钟 | 时间到进入最终判定 |
| **投票模式** | **非匿名** | 增加社交博弈深度 |
| **地点主题** | 可选 | 城市、科幻、历史等 |
