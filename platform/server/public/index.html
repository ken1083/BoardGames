    📜 寻宝之战 简明规则

    1. 阶段一：布设天罗地网 (PLACE X)
    在轮到你的回合时，首先必须在地图的任何一个空地上点一下以打下一枚路障(X)。注意：系统会严格把关，如果有人的路被你彻底封死了，此招将会被判定为非法死门！作为惩罚，这回合你将不许移动！

    2. 阶段二：长途跋涉 (MOVE)
    一旦成功施放阻碍，你可以控制你的角色光标向着周围八个方位（含斜对角）移动一格。

    3. 致胜真理 (WIN)
    每位玩家在右侧信息板都有专属分配的「暗金色宝藏点」。不要管别人，最先踩进那个格子的人立刻登基获胜！
