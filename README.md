# Online Boardgame Platform

这是我们的专属“在线聚会桌游平台”。

**项目愿景**：一切从简。这是一个极致轻量级的 Web Game 平台，让你和朋友们在面基或语音连麦时，只需要在电脑/手机浏览器里点开网站，一键建房、输入昵称，马上就能在一起斗智斗勇。它将是完全免费的、无需注册账号、也没有任何烦人的广告。

---

## 📁 游戏库与开发目录结构 (Directory Structure)

考虑到未来平台会有各种类型的桌游被添加进来，我们事先预设好“平台引擎基础”与“各游戏插槽”完全独立的目录形式。每一个新游戏都会被存放在它自己专属的 `games/<game_name>` 文件夹下。

```text
/OnlineGame
├─ docs/                  # [文档] 项目整体的开发流程、前后端技术栈与美学设计思路。
├─ platform/              # [大厅平台层] 核心的框架代码（待建）。
│  ├─ web/                # 通用的主页 Game List、加入房间 Lobby UI 等公共前端。
│  └─ server/             # WebSocket 连接池、全局所有房间的生命周期节点管理。
└─ games/                 # [游戏陈列室] 每个具体的游戏被收录在此。
   └─ treasure_hunt/      # 🎖️ 首发游戏项目：寻宝之战
      ├─ README.md        # =======> 当前游戏的介绍与核心规则（必读）
      ├─ assets/          # (预留) 存放本游戏可能专属的棋子、音效等独立静态资源。
      ├─ client/          # (预留) 专注渲染地图并实现本游戏界面的 React/UI 代码。
      └─ server/          # (预留) 专门校验本游戏打叉法则、胜利判定的独立后端逻辑块。
```
